*** Bindings

|---------+--------------------------------------------+-------------------------------------|
| C-1     | goe-describe-current-symbol                | Show doc for current symbol         |
| C-2     | goe-describe-current-func (only type info) | Show type info for current function |
| C-3     | goe-describe-current-func                  | Show doc for current function       |
